Documentação do Sistema Amoracoin

Introdução

Amoracoin é um sistema de moeda digital inovador, desenvolvido por Ravy Novais Sales em 24/12/2023, projetado para facilitar trocas online sem depender de dinheiro convencional. Diferentemente de muitas criptomoedas, a Amoracoin adota uma abordagem única que não se baseia em tecnologia blockchain. Esta moeda digital oferece uma alternativa eficiente e simplificada para transações online, promovendo uma adoção mais ampla de moedas digitais em diferentes setores.

O modelo da Amoracoin permite transações rápidas e seguras, com custos reduzidos, utilizando um sistema inovador baseado em um registro centralizado seguro para gerenciar transações e saldos de contas. Isso garante a integridade dos dados e a segurança das transações, eliminando a dependência do dinheiro físico e abrindo as portas para transações online sem barreiras.

As principais funcionalidades da biblioteca Amoracoin incluem login, criarConta, saldo, Minerar, transferirAmoracoin e outras funcionalidades essenciais para interagir com o sistema.

Aviso

O sistema Amoracoin ainda está em fase de teste e desenvolvimento. As informações aqui apresentadas podem sofrer alterações à medida que o projeto evolui.

Métodos

login

  • Propósito: Loga um usuário no sistema Amoracoin.
  • Uso:
Amoracoin.login(email, senha)
    .then(usuario => {
      // Usuário logado com sucesso
    })
    .catch(erro => {
      // Tratar erro de login
    });

criarConta

  • Propósito: Cria uma nova conta de usuário no sistema Amoracoin.
  • Uso:
Amoracoin.criarConta(email, senha, usuario)
    .then(() => {
      // Conta criada com sucesso
    })
    .catch(erro => {
      // Tratar erro de criação de conta
    });

logout

  • Propósito: Desconecta o usuário atualmente logado.
  • Uso:
Amoracoin.logout()
    .then(() => {
      // Usuário desconectado com sucesso
    })
    .catch(erro => {
      // Tratar erro de logout
    });

obterNomeUsuario

  • Propósito: Recupera o nome de usuário do usuário atualmente logado.
  • Uso:
Amoracoin.obterNomeUsuario()
    .then(usuario => {
      // Usar o nome de usuário
      console.log(usuario)
    })
    .catch(erro => {
      // Tratar erro ao buscar nome de usuário
    });

saldo

  • Propósito: Recupera o saldo atual de Amoracoins para o usuário logado.
  • Uso:
Amoracoin.saldo()
    .then(saldo => {
      // Usar o saldo
    })
    .catch(erro => {
      // Tratar erro ao recuperar saldo
    });

Minerar

  • PropósitoMinerar o terreno para ganhar os Amoracoins
  • Uso:
Amoracoin.Minerar()
    .then(mineracaoBemSucedida => {
      if (mineracaoBemSucedida) {
        // Mineração bem-sucedida
      } else {
        // Mineração sem sucesso
      }
    })
    .catch(erro => {
      // Tratar erro de mineração
    });

obterChave

  • Propósito: Recupera a chave de transferência para o usuário logado.
  • Uso:
Amoracoin.obterChave()
    .then(chaveTransferencia => {
      // Usar a chave de transferência
    })
    .catch(erro => {
      // Tratar erro ao recuperar a chave de transferência
    });

Autenticacao

  • Propósito: Recupera o status de autenticação atual do usuário e as informações do usuário.
  • Uso:
Amoracoin.Autenticacao()
    .then((userData) => {        
        // Aqui você pode usar os dados do usuário
        const { usuario, saldo, chaveTransferencia } = userData;
        console.log(`Usuário: ${usuario}, Saldo: ${saldo}, Chave de Transferência: ${chaveTransferencia}`);
    })
    .catch((error) => {
        console.error("Erro ao autenticar usuário:", error);
        // Trate o erro de acordo com a sua aplicação
    });

atualizarNomeUsuario

  • Propósito: Atualiza o nome de usuário do usuário atualmente logado.
  • Uso:
const novoNomeUsuario = "novonomeusuario";

Amoracoin.atualizarNomeUsuario(novoNomeUsuario)
  .then(() => {
    alert('Nome de usuário atualizado com sucesso!');
    // Atualizar a interface ou fazer outras operações necessárias
  })
  .catch((error) => {
    alert('Erro ao atualizar nome de usuário: ' + error);
  });

transferirAmoracoin

  • Propósito: Transfere Amoracoins do usuário atual para outro usuário.
  • Uso:
Amoracoin.transferirAmoracoin(valor, chaveDestinatario, chaveRemetente);

Métodos novos da versão 4.0

TotalDeAmoracoin

  • Propósito: Recupera o total de Amoracoin existente.
  • Uso:
Amoracoin.TotalDeAmoracoin()
    .then((total) => {
        console.log("Total de Amoracoin existente:", total);
    })
    .catch((error) => {
        console.error("Erro:", error);
    });

ChaveAnalisar

  • Propósito: Analisa uma chave de transferência e retorna o nome de usuário associado.
  • Uso:
const chave = 'chave_de_transferencia_aqui'; // Substitua pela chave desejada
Amoracoin.ChaveAnalisar(chave)
    .then((usuario) => {
        console.log("Nome de usuário:", usuario);
        alert("Nome de usuário: " + usuario); // Mostra o nome de usuário em um alerta
    })
    .catch((error) => {
        console.error("Erro ao analisar a chave:", error);
        alert("Erro ao analisar a chave: " + error);
    });

CongelarAmoracoins

  • Propósito: Congela um saldo específico de Amoracoin.
  • Uso:
Amoracoin.CongelarAmoracoins({
    nome: "meu_saldo",
    saldo: 100 // saldo a ser congelado
});

DescongelarAmoracoins

  • Propósito: Descongela um saldo específico de Amoracoin a partir de um arquivo.
  • Uso:
const arquivo = fileInput.files[0]; // Selecionar o arquivo desejado

Amoracoin.DescongelarAmoracoins(arquivo)
    .then(saldoRecuperado => {
        console.log(`Saldo de Amoracoin recuperado: ${saldoRecuperado} AMCR`);
        alert(`Saldo de Amoracoin recuperado: ${saldoRecuperado} AMCR`);
    })
    .catch(error => {
        console.error("Erro ao descongelar arquivo:", error);
        alert("Erro ao descongelar arquivo: " + error);
    });
