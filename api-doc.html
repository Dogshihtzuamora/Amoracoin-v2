Documentação da API - Amoracoin

Bem-vindo à documentação da API para a criação e consulta de carteiras da Amoracoin.

1. Criar uma nova carteira

Para criar uma nova carteira, envie uma requisição GET para o seguinte endpoint:

Endpoint:

https://amoracoin-serve.vercel.app/Wallet?create

Este endpoint irá gerar um novo endereço de carteira e retornar os detalhes necessários.

Nota: Não se esqueça de garantir a segurança ao usar a API, especialmente ao criar e consultar carteiras. A API está em constante evolução, então verifique sempre os endpoints para atualizações.

2. Consultar carteiras existentes

Para visualizar as carteiras existentes, você pode acessar o seguinte endpoint:

Endpoint:

https://amoracoin-serve.vercel.app/Wallet

Esse endpoint retorna uma lista das carteiras criadas, permitindo que você acompanhe os endereços registrados.

3. Realizar uma transferência

Para realizar uma transferência, utilize o seguinte endpoint:

Endpoint:

https://amoracoin-serve.vercel.app/transfer?private_key=${privateKey}&destination_address=${destinationAddress}&amount=${amount}

Substitua os seguintes parâmetros:

  • privateKey: A chave privada da carteira de origem
  • destinationAddress: O endereço da carteira de destino
  • amount: O valor a ser transferido

Taxa de Transferência

Para verificar a taxa atual de transferência, utilize o seguinte endpoint:

Endpoint:

https://amoracoin-serve.vercel.app/transfer?fee

Esse endpoint retorna a taxa de transferência vigente para transações na rede Amoracoin.

Importante: Para cada transação realizada, será cobrada uma taxa de 0.00001 amora. Esta taxa pode variar dependendo de alguns fatores. Certifique-se de ter saldo suficiente para cobrir tanto o valor da transferência quanto a taxa.

4. Procurar uma carteira

Para procurar uma carteira específica, você pode utilizar a seguinte URL com a chave privada:

Endpoint:

https://amoracoin-serve.vercel.app/Wallet?key_private=${suaKey}

Substitua ${suaKey} pela sua chave privada.

Importante: Nunca compartilhe sua chave privada com outras pessoas. Para realizar qualquer transação, é essencial manter sua chave privada segura. Caso contrário, sua carteira pode ser comprometida.

5. Mineração de Amoracoin

Para minerar Amoracoins e adicionar à sua carteira, utilize o seguinte endpoint:

Endpoint:

https://amoracoin-serve.vercel.app/mine?private_key=${privateKey}&seed=${seed}

Substitua os seguintes parâmetros:

  • privateKey: A chave privada da sua carteira
  • seed: Um número que será validado pelo sistema para verificar se você receberá a recompensa

Nota: A validação da seed determina se você receberá ou não a recompensa de mineração. O sistema utiliza um algoritmo de hash duplo com condições específicas para validar a seed fornecida.

Importante: A mineração está sujeita à disponibilidade total de Amoracoins no sistema. O valor da recompensa por mineração pode variar.

6. Excluir uma carteira

Para excluir uma carteira, utilize o seguinte endpoint:

Endpoint:

https://amoracoin-serve.vercel.app/Wallet?delete_wallet=${privateKey}

Substitua ${privateKey} pela chave privada da carteira que deseja excluir.

Importante: A exclusão de uma carteira é irreversível. Certifique-se de que deseja realmente removê-la antes de prosseguir.

Endpoint:
