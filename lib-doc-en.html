The Amoracoin library provides a simple and intuitive interface to interact with the Amoracoin blockchain. Here you will find everything you need to get started with our technology.

Importing the Library

To use the Amoracoin library, you need to import the JavaScript file into your HTML:

<script src="https://amoracoin.vercel.app/Src/Amoracoin lib/1.0.js"></script>

Then, you can use the library functions.

üîë Create Wallet

const wallet = Amoracoin.CreateWallet();

Return

  • address: Your unique address on the Amoracoin network
  • balance: Your current balance in amcr
  • privateKey: Your private key (keep it secure!)
  • created_at: The creation date of the wallet

It will create a new wallet or return an existing one saved in local storage.

üîç Find Wallet
