API Documentation - Amoracoin

Welcome to the API documentation for creating and managing wallets of Amoracoin.

1. Create a New Wallet

To create a new wallet, send a GET request to the following endpoint:

Endpoint:

https://amoracoin-serve.vercel.app/Wallet?create

This endpoint will generate a new wallet address and return the necessary details.

Note: Be sure to ensure security when using the API, especially when creating and querying wallets. The API is constantly evolving, so always check the endpoints for updates.

2. Query Existing Wallets

To view existing wallets, you can access the following endpoint:

Endpoint:

https://amoracoin-serve.vercel.app/Wallet

This endpoint returns a list of created wallets, allowing you to track the registered addresses.

3. Make a Transfer

To make a transfer, use the following endpoint:

Endpoint:

https://amoracoin-serve.vercel.app/transfer?private_key=${privateKey}&destination_address=${destinationAddress}&amount=${amount}

Replace the following parameters:

  • privateKey: The private key of the source wallet
  • destinationAddress: The address of the destination wallet
  • amount: The amount to be transferred

Transfer Fee

To check the current transfer fee, use the following endpoint:

Endpoint:

https://amoracoin-serve.vercel.app/transfer?fee

This endpoint returns the current transfer fee for transactions on the Amoracoin network.

Important: A fee of 0.00001 amora will be charged for each transaction. This fee may vary depending on several factors. Make sure you have enough balance to cover both the transfer amount and the fee.

4. Search for a Wallet

To search for a specific wallet, you can use the following URL with the private key:

Endpoint:

https://amoracoin-serve.vercel.app/Wallet?key_private=${yourKey}

Replace ${yourKey} with your private key.

Important: Never share your private key with others. To perform any transaction, it is essential to keep your private key secure. Otherwise, your wallet could be compromised.

5. Mine Amoracoin

To mine Amoracoins and add them to your wallet, use the following endpoint:

Endpoint:

https://amoracoin-serve.vercel.app/mine?private_key=${privateKey}&seed=${seed}

Replace the following parameters:

  • privateKey: The private key of your wallet
  • seed: A number that will be validated by the system to check if you will receive the reward

Note: The seed validation determines if you will receive the mining reward or not. The system uses a double hash algorithm with specific conditions to validate the provided seed.

Important: Mining is subject to the total availability of Amoracoins in the system. The reward for mining may vary.

6. Delete a Wallet

To delete a wallet, use the following endpoint:

Endpoint:

https://amoracoin-serve.vercel.app/Wallet?delete_wallet=${privateKey}

Replace ${privateKey} with the private key of the wallet you want to delete.

Important: Deleting a wallet is irreversible. Make sure you really want to remove it before proceeding.

8. Wallet Backup and Restoration

To ensure the integrity of your wallet, you can create a backup and restore it later. The backup process temporarily removes the wallet from the network, and you can reintroduce it to the network when necessary.

Create a Wallet Backup

To create a backup of your wallet, simply access the following endpoint, replacing ${privateKey} with your wallet's private key:

Endpoint:

https://amoracoin-serve.vercel.app/backup?key_private=${privateKey}

This endpoint will generate a backup of the wallet, temporarily removing it from the network. You can use this backup to restore the wallet later.

Restore a Wallet

To restore the wallet, use the following endpoint, passing the backup content in the URL:

Endpoint:

https://amoracoin-serve.vercel.app/backup?restore=${content}

Replace ${content} with your backup content. The restoration will reintroduce the wallet to the network, allowing you to access your funds again.

Note: The backup temporarily removes the wallet from the network. Make sure to keep the backup secure to prevent any loss of important data.

Important: Restoring a wallet may overwrite any changes made after the backup. Make sure the restoration is necessary before proceeding.
